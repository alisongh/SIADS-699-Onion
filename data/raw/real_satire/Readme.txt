Satirical News Detection and Analysis using Attention Mechanism and Linguistic Features
Fan Yang, Arjun Mukherjee, Eduard Gragut
=========================================================================================
This folder contains the data collection of satirical news and true news.
Within each file, the document is separated by "******"; Each line is a paragraph.
This data collection doesn't contain author information, headline, or creation time. 
Only news articles are provided for the moment.
